# Contributing to Cloud Drive Integration

First off, thank you for considering contributing to Cloud Drive Integration! It's people like you who make the Linux ecosystem better for everyone.

## Code of Conduct

This project and everyone participating in it is governed by basic principles of respect and collaboration:
- Be welcoming and friendly
- Be patient and understanding
- Be respectful of differing viewpoints
- Focus on what is best for the community
- Show empathy towards other community members
